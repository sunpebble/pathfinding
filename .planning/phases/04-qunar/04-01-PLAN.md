  <files>apps/ai-service/src/lib/crawlers/accessibility-parser.ts</files>
  <action>
Add `transformToHighResQunar()` function that transforms Qunar CDN URLs to high-resolution versions.

Qunar uses different CDN patterns than Ctrip. Common patterns:

- `qnimg.qunar.com` - Qunar's image CDN
- `p0.meituan.net`, `p1.meituan.net` - Sometimes uses Meituan CDN
- `dimg.qunarzz.com` - Another Qunar CDN domain

Transformation rules:

1. For qnimg/qunarzz URLs: Remove size constraints like `_r_` or `_w_` parameters
2. For thumbnail suffixes: Remove `_small`, `_thumb`, `_s` suffixes
3. For query params limiting size: Remove or modify `w=`, `h=`, `q=` params

If URL doesn't match Qunar CDN patterns, return unchanged.

Function signature: `transformToHighResQunar(url: string): string`

Export the function.
</action>
