Add two more exported functions to accessibility-parser.ts:

1. **extractPublishDate(content: string): string | undefined**
   - Parse dates in formats:
     - ISO-like: /(\d{4}-\d{2}-\d{2})/
     - Chinese: /(\d{4})年(\d{1,2})月(\d{1,2})日/
     - With label: /发布(?:时间)?[：:]\s\*(\d{4}-\d{2}-\d{2})/
   - For Chinese format, pad month/day to 2 digits
   - Return undefined if no match
   - Return first match found (priority: labeled > ISO > Chinese)

2. **extractCtripStats(content: string): { views?: number; likes?: number; saves?: number; comments?: number }**
   - Use parseChineseNumber() internally for all numeric parsing
   - Views: match /(?:阅读|浏览)量?\s*([\d.]+\s*[千万kw]?)/i
   - Likes: match /(?:点赞|个赞|赞)\s*([\d.]+\s*[千万kw]?)/i OR /([\d.]+\s*[千万kw]?)\s*(?:点赞|个赞)/i
   - Saves: match /收藏\s*([\d.]+\s*[千万kw]?)/i OR /([\d.]+\s*[千万kw]?)\s*收藏/i
   - Comments: match /(?:评论|条评论)\s*([\d.]+\s*[千万kw]?)/i OR /([\d.]+\s*[千万kw]?)\s*(?:条评论|评论)/i
   - Return empty object properties as undefined, not 0

Place these after the functions from Task 1.

Export all 4 new functions in the module exports.
</action>
