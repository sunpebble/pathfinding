# Implementation Plan: 出行攻略 (Travel Itinerary)

**Branch**: `001-travel-itinerary` | **Date**: 2026-01-02 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/001-travel-itinerary/spec.md`

## Summary

Travel itinerary planning feature enabling users to create, edit, and manage trip plans with a timeline-based interface. Supports POI recommendations, offline editing, one-click itinerary copying from community, and integration with third-party navigation/ride-hailing apps. Mobile-first design with React Native + Deno/Hono backend + Supabase.

## Technical Context

**Language/Version**: Deno (stable), TypeScript 5.x, React Native (Expo SDK 52)  
**Primary Dependencies**: Hono (backend), Expo Router (mobile), Supabase client, React Native Maps, React Native Reanimated (drag-drop)  
**Storage**: Supabase (PostgreSQL + Realtime), AsyncStorage + WatermelonDB (mobile offline cache)  
**Testing**: Deno test (backend), Jest + React Native Testing Library (mobile)  
**Target Platform**: iOS 14+, Android 8+, Deno runtime on server  
**Project Type**: Monorepo (mobile app + API backend + shared packages)  
**Performance Goals**: API < 500ms p95, mobile 60fps, < 2s itinerary load, offline sync < 5s  
**Constraints**: < 100KB API responses, < 50MB app bundle, offline support for 10 itineraries  
**Scale/Scope**: 10k+ users, 100k+ itineraries, offline-first with conflict resolution

## Complexity Tracking

> No constitution violations. All requirements align with established principles.

| Aspect           | Decision                | Rationale                                                  |
| ---------------- | ----------------------- | ---------------------------------------------------------- |
| Offline Storage  | WatermelonDB            | Better performance than AsyncStorage for relational data   |
| Drag-Drop        | React Native Reanimated | 60fps requirement, native driver for smooth animations     |
| State Management | Zustand                 | Simpler than Redux, works well with offline-first patterns |
