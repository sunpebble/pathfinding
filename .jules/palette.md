## 2024-02-28 - Adding Missing ARIA Labels

**Learning:** Adding `aria-label`s to icon-only buttons improves accessibility for screen readers and fulfills the Palette agent's mission to add micro-UX enhancements.
**Action:** Always check interactive elements (buttons, links) that only contain an icon for an `aria-label` or `title`. Use `<button aria-label="Action name">` for screen readers and `title="Action name"` for mouse hover tooltips.
